1. Which of these keywords must be used to handle the exception thrown by try block in some rational manner?
a) try
b) finally
c) throw
d) catch

Answer d

2. Which of these keywords is used to manually throw an exception?
a) try
b) finally
c) throw
d) catch

Answer c

3. User defined exceptions are extended from
a) Exception Class
b) Error Class
c) Try Class
d) Catch Class

Answer a

4. User defined exceptions are

a) Checked Exceptions
b) Unchecked Exceptions
c) Both a and b
d) None

Answer a

5. Which of the following is not Unchecked Exception
a. ArithmeticException
b. NullPointerException
c. ArrayIndexOutOfBoundsException
d. SQLException

Answer d
