1. Which of the following is the best type of module coupling?
a) Control Coupling
b) Stamp Coupling
c) Data Coupling
d) Content Coupling

Answer c

2. _______________ is a measure of the degree of interdependence between modules.
a) Cohesion
b) Coupling
c) None of the mentioned
d) All of the mentioned

Answer b

3. Which of the following is the worst type of module coupling?
a) Control Coupling
b) Stamp Coupling
c) External Coupling
d) Content Coupling

Answer c


4. Which of the following is the best type of module cohesion?
a) Functional Cohesion
b) Temporal Cohesion
c) Functional Cohesion
d) Sequential Cohesion

Answer a

5. A software engineer must design the modules with the goal of high cohesion and low coupling.
a) True
b) False
c) --
d) --

Answer a

6. In what type of coupling, the complete data structure is passed from one module to another?
a) Control Coupling
b) Stamp Coupling
